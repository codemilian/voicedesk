# VoiceDesk AI — permanent demo links

This site is a tiny redirector. Links like `/try/<slug>` forward to the
current VoiceDesk AI funnel, whose address lives in [config.json](config.json).

When the funnel moves (new tunnel, Render, custom domain), update
`config.json` — every link ever shared keeps working.

Managed automatically by `outreach/permalink.js` in the VoiceDesk repo.
